# HomeBrain User Guide

This is for household users, not the system administrator.

## Talking To HomeBrain

1. Say the wake word
2. Wait for the acknowledgment
3. Say the command

Examples:

- "Anna, turn on the kitchen lights."
- "Henry, run bedtime."
- "Anna, start movie night."
- "Henry, what devices are still on?"

## What You Can Usually Ask

- device control
- scene activation
- workflow execution
- simple home status questions

Exact capabilities depend on what the admin has configured.

## If It Does Not Respond

Try these basic checks first:

1. Move closer to the listener
2. Reduce background noise
3. Use the exact configured wake phrase
4. Ask the admin whether the listener is online

## Privacy Basics

HomeBrain is designed for local-first use:

- the hub runs inside your network
- listener devices stay on your network
- local speech-to-text can run on the hub

The admin may still enable cloud services such as OpenAI or ElevenLabs, so ask them if you want to know which providers are enabled.
